## 🧪 Модули системы

### src/config.py
Конфигурационные параметры:
- Параметры ЦФА (доходность, срок, объем)
- Критерии отбора облигаций
- Аналитические параметры (ставки, налоги)
- Настройки визуализации

### src/data_fetcher.py
Получение рыночных данных:
- Поиск облигаций на Мосбирже
- Детали по выпускам
- Рыночные данные (доходность, цены)
- Sample данные для демонстрации

### src/dfa_analyzer.py
Анализ цифровых финансовых активов:
- Расчет купонных платежей
- Эффективная доходность
- Risk-adjusted метрики (Sharpe ratio)
- Полный анализ ЦФА

### src/bonds_analyzer.py
Анализ облигаций:
- Эффективная доходность (YTM)
- Скорректированная на ликвидность доходность
- Рисковые метрики (beta, Sharpe ratio)
- Статистика по выборке

### src/comparator.py
Сравнительный анализ:
- Сравнение на первичном рынке
- Посленалоговое сравнение
- Анализ влияния ликвидности
- Декомпозиция спреда
- Инвестиционные рекомендации

### src/visualizer.py
Визуализация результатов:
- Bar charts (сравнение доходности)
- Horizontal bar plots (анализ спреда)
- Scatter plots (риск-доходность)
- Stacked bar (декомпозиция)
- Multi-panel dashboard

### src/report_generator.py
Генерация отчетов:
- Markdown отчеты
- Excel файлы с данными
- Outline для презентаций

## ⚠️ Ограничения

- **Данные по облигациям:** Попытка получить через MOEX API, fallback на sample данные
- **Риск-метрики:** Упрощены (нет реальных данных по волатильности)
- **Ликвидность:** Оценена экспертно (нужны реальные данные по оборотам)
- **MOEX API:** Может быть недоступен или ограничен (тогда используются sample данные)
